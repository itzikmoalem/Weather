This program use Ajax, The user can check weather.

Assumptions

This program is weather that the user needs to input Name, Longitude and Latitude. all the inputs checked by validations, and if it's ok, the input add to dynamic list. I made the weather by click on item on the list. (without "display" button) the output is weather with date, temperature, icon and wind speed (I did by km/s). this program contact with "http://www.7timer.info/doc.php?lang=en#graphical_api" server, and take the data by Json.
I chose to use Icons and Text to display the weather.
Design: I designed the head picture (HAC weather), and create my own icons.
I used namespace.
I used Ajax, JSON.
